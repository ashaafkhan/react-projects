# Jokes Viewer Application

Live demo: https://jokes-viewer-application.vercel.app/

## Overview
A lightweight React app that fetches jokes from an API and displays them in a clean, readable layout. Built with Vite for fast dev and production builds.

## Features
- Fetch jokes from a public API
- Simple, responsive layout
- Loading and error handling

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```

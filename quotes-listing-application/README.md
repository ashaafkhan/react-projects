# Quotes Listing Application

Live demo: https://quotes-listing-application-ten.vercel.app/

## Overview
A minimal quotes browser that pulls quotes from a public API and presents them in a clean, readable layout with author details.

## Features
- Fetch quotes from an external API
- Simple, readable card layout
- Loading and error states

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```

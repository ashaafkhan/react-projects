# Product Listing Interface

Live demo: https://product-listing-interface-pi.vercel.app/

## Overview
A simple product catalog UI that fetches product data from the FreeAPI Products endpoint and displays it in a clean card grid with pricing and metadata.

## Features
- Fetch products from the FreeAPI endpoint
- Responsive card layout
- Loading and error states
- Price, category, rating, and stock details

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```

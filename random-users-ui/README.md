# Random Users UI

Live demo: https://random-users-ui-kappa.vercel.app/

## Overview
A user directory UI that fetches random user profiles from a public API and displays them in a clean, responsive grid with key details.

## Features
- Fetch random user profiles
- Responsive card layout
- Loading and error states

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```

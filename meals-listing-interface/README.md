# Meals Listing Interface

Live demo: https://meals-listing-interface-pi.vercel.app/

## Overview
A clean meals catalog UI that fetches meal data from a public API and displays it in a responsive card grid with category, area, and recipe links.

## Features
- Fetch meals from an external API
- Responsive card layout
- Loading and error states
- Links to recipe videos when available

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```

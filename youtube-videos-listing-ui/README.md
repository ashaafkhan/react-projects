# YouTube Videos Listing UI

Live demo: https://youtube-videos-listing-livid.vercel.app/

## Overview
A YouTube-style video listing interface that fetches video data from a public API and displays it in a clean, responsive layout.

## Features
- Fetch video data from an external API
- Responsive card layout
- Loading and error states

## Tech Stack
- React
- Vite
- CSS

## Run Locally
```bash
npm install
npm run dev
```
